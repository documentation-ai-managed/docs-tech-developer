## Overview

Build a simple Flutter app to get started.

## Steps

<Steps>
  <Step title="Create new project">
    ````bash
    flutter create hello_flutter
    cd hello_flutter
    ````
  </Step>
  <Step title="Run the app">
    ````bash
    flutter run
    ````
  </Step>
</Steps>
