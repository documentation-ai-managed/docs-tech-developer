<Update label="2024-10-03" description="v3.24.0" tags={["feature", "improvement", "bugfix"]}>

## New Features

- Added `PlatformAdaptiveNavigationBar` for responsive bottom navigation across platforms.
- Introduced `CupertinoDynamicColor` for adaptive colors in iOS apps.

## Improvements

- Enhanced Impeller rendering engine stability on macOS and Linux.
- Reduced binary size by `>15%` for release builds.

## Bug Fixes

- Fixed `ListView` infinite scrolling issues on Android.
- Resolved hot reload failures with certain stateful widgets.

</Update>

<Update label="2024-08-29" description="v3.22.2" tags={["feature", "bugfix"]}>

## Upgrading Flutter

Follow these steps to update your Flutter installation and migrate your apps.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run the following command to verify your version:
    
    ````bash
    flutter --version
    ````
  </Step>
  <Step title="Update SDK" icon="download">
    Upgrade to the latest stable channel:
    
    ````bash
    flutter channel stable
    flutter upgrade
    ````
  </Step>
  <Step title="Run Migration" icon="settings">
    Use the built-in analyzer for breaking changes:
    
    ````bash
    flutter analyze
    dart migrate
    ````
  </Step>
  <Step title="Test Your App" icon="play-circle">
    Build and test on all target platforms:
    
    ````bash
    flutter test
    flutter build apk --release  # For Android
    ````
  </Step>
</Steps>

<Callout kind="alert">
  Review breaking changes in each release before upgrading production apps. Test thoroughly on physical devices.
</Callout>
